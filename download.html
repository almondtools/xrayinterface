Get XRayInterface

XRayInterface is available from the Maven repository.

XRayInterface requires Java 8, for lower versions consider Picklock, which is the predecessor of XRayInterface.

Download

  • xrayinterface-0.3.2.jar
  • xrayinterface-0.3.2-javadoc.jar
  • xrayinterface-0.3.2-sources.jar

Maven Dependency

<dependency>
    <groupId>net.amygdalum</groupId>
    <artifactId>xrayinterface</artifactId>
    <version>0.3.2</version>
</dependency>
