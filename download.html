Get XRayInterface

XRayInterface is available from the Maven repository.

XRayInterface requires Java 8, for lower versions consider Picklock , which is the predecessor of XRayInterface.

Download

  • xrayinterface-0.2.8.jar
  • xrayinterface-0.2.8-javadoc.jar
  • xrayinterface-0.2.8-sources.jar

Maven Dependency

<dependency>
    <groupId>com.github.almondtools</groupId>
    <artifactId>xrayinterface</artifactId>
    <version>0.2.8</version>
</dependency>
